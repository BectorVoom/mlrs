<objective>
Build the pure-Python `mlrs` shim (D-01): each of the 12 estimators is a sklearn
`BaseEstimator` + family-mixin subclass that does numpy/list/pyarrow->contiguous-pyarrow
normalization (D-02), `output_type` mirror routing (D-03), default-dtype selection
(D-05), and delegates compute to the Plan-03 `_mlrs` wrappers. `get_params`/`set_params`/
`clone`/`__repr__` come from sklearn for free (faithful `__init__`). `fit` returns `self`.

Purpose: This is the user-facing API surface and the thing `sklearn.utils.estimator_checks`
exercises (Plan 06). Interface-first: it consumes the uniform `_mlrs` low-level
classes from Plan 03 and the `backend_supports_f64()` flag from Plan 02.
Output: `base.py`, `_io.py`, the four family modules, `__init__.py`, and a `test_params.py`
that proves PY-02 (`get_params`/`set_params` round-trip + sklearn names).
</objective>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user numpy/list/pyarrow input -> shim normalization | malformed shapes / non-finite values must be rejected with sklearn-standard errors before crossing |
| shim fitted-attr access -> _mlrs accessor | accessing before fit must raise NotFittedError, not reach the unfitted wrapper |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-11 | Tampering | Sliced/non-contiguous numpy view crossing as an aliased pyarrow slice | mitigate | `normalize_X` does `np.ascontiguousarray(X).ravel()` -> a FRESH pyarrow array; never a zero-copy slice (Pitfall 3). The Rust bridge (Plan 02) is the backstop hard-reject. |
| T-06-12 | Tampering | NaN/inf input silently computed on | mitigate | `check_array(..., force_all_finite=True)` raises the sklearn-standard error before normalization (estimator_checks point 5); the bridge null-reject is the backstop. |
| T-06-13 | Information disclosure | Reading fitted attrs before fit | mitigate | `_check_fitted()` (`check_is_fitted`) raises `NotFittedError` before any `_mlrs` accessor call. |

No unmitigated `high` threats. Input validation happens in Python (sklearn
check_array) with the Rust bridge as the backstop.
</threat_model>
