<objective>
Implement the phase's first novel device routine: a one-sided Jacobi SVD (D-01) that
returns the thin/economy factors U (m×k), S (k), Vᵀ (k×n) with k=min(m,n) (D-02),
handles tall AND wide inputs via the Aᵀ-and-swap dispatch (D-05), sorts descending
(D-04), and validates standalone against numpy fixtures + reference-free invariants
(D-09) on cpu (f32+f64) and rocm (f32). The convergence sweep stays device-resident in
a single cube (D-11 gate 3).

Purpose: PCA, TruncatedSVD, and the OLS/Ridge SVD solver (four Phase-4 estimators)
consume this primitive; a convention bug here propagates to all four, so the
fixture + basis-invariant double-check is load-bearing. This is the hardest single
piece of device code in the project.
Output: jacobi_svd.rs kernel, prims/svd.rs host orchestration, and the green SVD
oracle + invariant test suite (the plan-02 scaffold's #[ignore]s removed).
</objective>
