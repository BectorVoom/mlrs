    1. `gaussian_matrix::<F>(pool, seed, n_components, n_features) -> Result<DeviceArray<..,F>, PrimError>`:

       each entry is 0 w.p. (1−density); +v w.p. density/2; −v w.p. density/2; where
       `v = sqrt((1/density)/n_components)` (RESEARCH Pattern 4 exact scaling). Draw the
       branch from `next_f64()`. Store DENSE (D-12, no sparse device kernel). Single upload.

    3. `permutation(seed, n) -> Vec<usize>`: Fisher-Yates over `0..n` using
       `SplitMix64::next_below` (UNBIASED rejection sampling) for each swap index — NEVER
       `next_u64() % n` (RESEARCH Anti-Pattern "biased modulo").

    Validate-before-allocate (ASVS V5): reject `n_components == 0` / `n_features == 0` as
    `PrimError::ShapeMismatch` and `density ∉ (0,1]` as `PrimError::ShapeMismatch` (synthetic
