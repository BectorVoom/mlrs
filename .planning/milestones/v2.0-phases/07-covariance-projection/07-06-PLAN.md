Implement PROJ-01 (`GaussianRandomProjection`) and PROJ-02 (`SparseRandomProjection`)
as mlrs-algos transformers composing the PRIM-06 rng prim (Plan 02) for the projection
matrix and the v1 gemm for transform. `n_components='auto'` resolves via
`johnson_lindenstrauss_min_dim` (the ONE value-matched quantity, D-12). Correctness is
PROPERTY-gated (D-12): JL distortion bound (averaged over trials, strict per D-10),

MT19937). Strict bands stay reproducible via a fixed seed + averaging (D-11).

matrix DENSE (D-12, no sparse device kernel in v2); sparse input is densified at the
Python ingress (PROJ-02 / handled in the PyO3 wrapper, Plan 07).
Output: projection/gaussian.rs (+ jl_min_dim), projection/sparse.rs,
random_projection_test.rs green (property gate + jl_min_dim value oracle).
