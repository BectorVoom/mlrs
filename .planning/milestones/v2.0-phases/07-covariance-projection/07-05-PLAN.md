<objective>
Implement DECOMP-03 (`IncrementalPCA`) as an mlrs-algos estimator implementing the new
`PartialFit<F>` trait (D-01) and composing the PRIM-07 incremental_svd merge (Plan 03).
`fit()` is sklearn-faithful (D-02): it resets state then loops `partial_fit` over
`gen_batches(n_samples, batch_size)` with `batch_size = 5·n_features` when None (D-03).
`whiten` (D-06) scales components by 1/sqrt(explained_variance_) in transform and
un-whitens in inverse_transform. All attrs + transform/inverse_transform match sklearn
within 1e-5 after align_rows sign alignment.

Purpose: the streaming-decomposition estimator. It mirrors the v1 pca.rs skeleton
(struct, accessors, Transform/inverse_transform) but the FIT path is the multi-batch
merge instead of a single SVD. The explained_variance ddof=1 here is DISTINCT from the
covariance estimators' ddof=0 (Pitfall 1).
Output: decomposition/incremental_pca.rs, incremental_pca_test.rs green at 1e-5 (f64)
via partial_fit AND via fit(), whiten on/off.
</objective>
