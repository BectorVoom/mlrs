<objective>
Implement COV-01 (`EmpiricalCovariance`) and COV-02 (`LedoitWolf`) as mlrs-algos
estimators composing the validated v1 `covariance` (ddof=0) and `eig` (pinvh) prims.
EmpiricalCovariance: `covariance_` = MLE Gram (ddof=0, == np.cov(bias=1)), `location_`
= mean (or 0 if assume_centered, D-07), `precision_` = eig-based pinvh (D-05/D-08,
singular-safe). LedoitWolf: the exact sklearn 1.7.1 `ledoit_wolf_shrinkage` β/δ/μ
closed form on top of the same ddof=0 emp_cov, `shrinkage_` clipped to [0,1].

Purpose: the covariance family. Both estimators are host orchestration over existing
device prims — no new prim, no new kernel. The pinvh (eig, not Cholesky) is the
load-bearing D-05 decision: it handles the rank-deficient MLE case (n ≤ p) without
raising, which Cholesky cannot.
Output: covariance/empirical_covariance.rs, covariance/ledoit_wolf.rs, two oracle
tests green at 1e-5 (f64) incl. a rank-deficient precision_ case.
</objective>
