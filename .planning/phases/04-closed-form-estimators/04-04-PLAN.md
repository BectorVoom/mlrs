<objective>
Deliver the two decomposition estimators (DECOMP-01 `PCA`, DECOMP-02
`TruncatedSVD`) — both built on the SAME Phase-3 thin SVD + `align_rows`
skeleton, with three documented per-estimator differences. PCA = SVD of centered
X (D-01, NOT eig-of-covariance), `explained_variance_ = S²/(n−1)`. TruncatedSVD =
SVD of UNCENTERED X, `explained_variance_ = var(transform(X) columns)`, oracle =
deterministic `algorithm='arpack'` (D-07). `svd_flip` is applied by the estimator
via `align_rows`; the primitive stays raw (D-01/D-03).

These two estimators share a code skeleton and consume only validated primitives
(no convergence risk). They edit only `decomposition/` files (04-01 owns
`lib.rs`), so this plan runs parallel with LinearRegression (04-03).

Purpose: complete the decomposition surface and prove the SVD-of-centered-X PCA
arithmetic and the distinct TruncatedSVD variance formula against sklearn.
Output: `Pca` + `TruncatedSvd` structs; activated `pca_test.rs` +
`truncated_svd_test.rs`.
</objective>
