<objective>
Complete the host tree back-end: condense the single-linkage hierarchy by
`min_cluster_size`, compute stabilities, run the eom/leaf/epsilon/max_cluster_size
cluster selection, label points (`-1`=noise) and compute `probabilities_`. After this
plan the precomputed path produces exact labels (HDBS-02) and ≤1e-5 probabilities
(HDBS-01) — the full host back-end is proven before the device front-end (Wave 4) feeds
it feature-metric data.

Purpose: this is the rest of the sklearn `_tree.pyx` port. It must match sklearn's
EoM/leaf traversal and labelling bit-for-bit so the exact-label gate holds for every
selection knob (D-09).
Output: condense.rs, stability.rs, select.rs, and the precomputed `fit` producing
labels + probabilities.
</objective>
