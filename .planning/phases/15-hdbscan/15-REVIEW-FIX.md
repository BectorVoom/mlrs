## Fixes Applied

### WR-01 ‚Äî `clamp(1, n)` panic when `n == 0` ‚Äî FIXED (`0c527cd`)
Guarded `n == 0` in `core_distances_dense` (early return); documented the
`feature_metric_single_linkage` clamp invariant where `validate_geometry`
guarantees `n >= 1`.

### WR-02 ‚Äî `get_clusters` infers `n_samples == 0` from singleton rows ‚Äî FIXED (`647fe5b`)
Threaded the true point count from `tree_to_labels` into `get_clusters` instead
of reconstructing it from singleton rows (mirrors sklearn passing `num_points`
explicitly).

### WR-03 ‚Äî `centroids_`/`medoids_` returns `Some(empty)` vs documented `None` ‚Äî FIXED (`2226dab`)
Mapped empty centroid/medoid results to `None` to honor the documented
`None`-means-no-cluster contract.

### WR-04 ‚Äî asymmetric precomputed matrix silently wrong MST ‚Äî FIXED (`cc5d51b`)
Added a `debug_assert!` near-symmetry check on the precomputed distance matrix
before computing core distances, catching fixture regressions in test builds.

### WR-05 ‚Äî recursive `find`/`recurse_leaf_dfs`/`traverse_upwards` stack overflow ‚Äî FIXED (`93f9789`)
Converted all three to iterative implementations (two-pass `find`, explicit-stack
DFS/upward traversal) to eliminate the stack-overflow risk on deep/degenerate
trees. Oracle suite 40/40.

### WR-06 ‚Äî `weighted_cluster_center` assumes dense `0..k` label range ‚Äî FIXED (`68cd822`)
Added an assertion of the dense `0..n_clusters` label-range invariant so a sparse
range fails loudly rather than emitting phantom all-zero center rows.

### IN-01 ‚Äî `usize::MAX` fabricated `len` sentinel in overflow errors ‚Äî FIXED (`4662fe9`)
Added a dedicated `PrimError::Overflow` variant and replaced the `usize::MAX`
sentinels in the `checked_mul` overflow guards.

### IN-02 ‚Äî `nn` bound only for a release-stripped `debug_assert_eq!` ‚Äî FIXED (`7c27126`)
Documented that the `checked_mul` guard (not the debug assert) is the
load-bearing element, clarifying the `nn` binding is not dead code in release.

### IN-03 ‚Äî `host_pairwise` duplicated across two modules ‚Äî FIXED (`aa0b5ca`)
Extracted a single shared `host_pairwise` (with the `Cosine` arm) into a new
`hdbscan/distance.rs` submodule; both callers now use it. Oracle suite 40/40.

### IN-04 ‚Äî `gen_oracle.py` is build-time-only ‚Äî NO ACTION REQUIRED
The review prescribed "Fix: None". The script regenerates committed `.npz`
fixtures, is never run in CI/tests, and cannot affect shipped behavior. No
defect; left unchanged.

### IN-05 ‚Äî `do_labelling` over-sizes then truncates with defensive `max` ‚Äî FIXED (`a9235ef`)
Replaced the defensive `root_cluster.max(n_samples)` sizing with an explicit
`debug_assert_eq!(root_cluster, n_samples)` invariant and sized the result to
`n_samples` directly. Oracle suite 40/40.
