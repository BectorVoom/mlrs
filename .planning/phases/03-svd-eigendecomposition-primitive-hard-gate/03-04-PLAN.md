<objective>
Implement the phase's second distinct routine (D-01): a classic two-sided cyclic Jacobi
symmetric-eigendecomposition for the covariance/Gram PCA-`full` path. It validates
squareness and trusts symmetry (D-06), returns eigenvalues/eigenvectors sorted descending
(D-04), reuses the covariance/GEMM output buffer rather than allocating a parallel matrix
(D-11 gate 2), keeps the convergence sweep device-resident (D-11 gate 3), and validates
against np.linalg.eigh + the eig-residual invariant (D-09) on cpu (f32+f64) and rocm (f32).

Purpose: PCA's `full` solver path consumes this directly; returning true signed
eigenvalues (rather than deriving them from the SVD) is why D-01 chose a separate routine.
It runs in Wave 3 because it shares mlrs-kernels/src/lib.rs and prims/mod.rs with Plan 03,
and it reuses the SVD kernel's now-proven shared-memory / sync_cube / round-robin idioms.
Output: jacobi_eig.rs kernel, prims/eig.rs host orchestration, and the green eig oracle +
residual-invariant test suite (the plan-02 scaffold's #[ignore]s removed).
</objective>
